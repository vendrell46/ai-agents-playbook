# AI Agents Playbook

Real-world AI agent architectures, built and running in production.

This repo documents autonomous agent systems I've built and operate daily to run [Zealynx Security](https://zealynx.io) — a smart contract audit firm. These aren't demos or tutorials. They're production systems running 24/7, handling business development, content, lead generation, personal health tracking, and security tooling.

## Systems

| System | What It Does | Status |
|--------|-------------|--------|
| [OpenClaw Ops](openclaw-ops/) | 6-agent autonomous business operations platform | Production (40 days) |

## Philosophy

- **Build with AI, not just about AI.** Every system here runs a real business function — not a proof of concept.
- **Measure everything.** If it doesn't have metrics, it doesn't exist.
- **Fail transparently.** Every system section includes what broke and what I learned.
